鉱石採掘クリッカーです

やり方

旗を押す　スペースキー　真ん中の鉱石をクリック！

村人を押すと色々買えます　ホッパーは自動でコインがとれるように！

鉱石を買い、左下のCボタンを押すと変えられます

今後もいろいろな機能を追加していく予定です。
